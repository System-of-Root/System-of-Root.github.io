Installing Mods for Arrow a Row

Melon Loader

Mods for Arrow a Row use the Melon Mod Loader v0.5.7 The installer for that can be downloaded here: MelonLoader.Installer.exe

Before installing the loader, You will need to make a copy of the game files. This is because the game's default installation path is incompatible with Melon. Start by secondary-clicking (Right-click on most computers) on the game in Steam and selecting Manage > Browse local files.

This should open File Explorer to a path similar to "SteamLibrary\steamapps\common\箭箭剑 Arrow a Row." You need to back out to the "common" folder and copy the game's install folder somewhere (it does not really matter where.) Rename the copy to something that does not have non-English characters in it. For example, "Modded Arrow A Row"

The game is now ready to have the mod loader installed. Open the installer you downloaded and click the SELECT button. Navigate to where you copied the game, and select "Arrow a Row.exe," uncheck the Latest check box, select version 0.5.7, and click install. Your settings should look like this:

Installing Mods

The installer should have created a "Mods" folder inside of the game's folder. Mods are installed into this folder. To install a mod, copy all included files into the Mods folder Note: it is essential that the mod files sit directly inside that folder and not in a sub-folder.

Once all the mods you wish to play with are installed, open the Arrow a Row.exe. If you have done everything correctly, you should see a small console open, then MelonLoader launching the game proper.
