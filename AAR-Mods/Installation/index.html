Installing Mods for Arrow a Row

Mellon Loader

Mods for Arrow a Row use the Melon Mod Loader v0.5.7 The installer for that can be downloaded here: MelonLoader.Installer.exe

Before we install the loader however, we need to make a copy of the game files. This is because the default Installation path for the game is incompatible with Mellon. Start by secondary-clicking (Right click on most computers) on the game in steam and selecting Manage > Browse local files.

This should open file explorer to a path similar to "SteamLibrary\steamapps\common\箭箭剑 Arrow a Row" you need to back out to the "common" folder and copy the game's install folder somewhere (it dose not really matter where.) Then rename the copy to something that dose not have non-english characters in it. For example "Modded Arrow A Row"

We are now ready to install the mod loader. Open the installer you downloaded and click the SELECT button. Then navigate to where you copied the game to, and select "Arrow a Row.exe" then uncheck the Latest check box and select version 0.5.7 and click install. Your settings should look like this:

Installing Mods

The installer should have created a "Mods" folder inside of the game's folder. This is where you will install all of your mods. To do this, simply copy the mod files into that folder. Note: It is important that the mode files sit directly inside that folder and not in a sub folder

Once all of the mods you wish to play with are installed, simply open the Arrow a Row.exe if you have done everything correctly you should see a small console open and then MelonLoader lunching the game proper.
